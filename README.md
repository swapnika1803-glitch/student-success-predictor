Student Success Predictor

This project predicts a student's final marks based on multiple factors using Machine Learning and Streamlit.

Features

- Study Hours
- Sleep Hours
- Attendance Percentage
- Previous Marks
- Predicts Final Marks using Linear Regression

Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit

How to Run

1. Install required libraries:

pip install streamlit pandas numpy scikit-learn

2. Run the application:

streamlit run app.py

3. Open the browser at:

http://localhost:8501

Project Description

This project demonstrates a simple Machine Learning model that analyzes student habits and predicts academic performance. It helps understand how different factors affect student success.
