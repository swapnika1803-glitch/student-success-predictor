# Student Success Predictor

This project predicts whether a student will Pass or Fail based on:
- Study Hours
- Sleep Hours
- Attendance
- Previous Marks

## Model
Logistic Regression (Machine Learning)

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## How to Run

Install libraries:
pip install streamlit pandas scikit-learn numpy

Run the app:
streamlit run app.py
